A modern, animated developer portfolio website built with React, Tailwind CSS, and Framer Motion. Showcases skills, projects, and provides an easy way to connect via WhatsApp.

## 🌟 Live Demo



## ✨ Features

- 🎨 **Modern Design** - Glass morphism effects with gradient backgrounds
- 🚀 **Smooth Animations** - Scroll-triggered animations using Framer Motion
- 📱 **Fully Responsive** - Perfect on all devices (mobile, tablet, desktop)
- 💼 **Project Showcase** - Display your best work with interactive cards
- 📝 **Contact Form** - Direct WhatsApp integration for messages
- 🧠 **Skills Section** - Organized tech stack presentation
- 🎯 **SEO Friendly** - Optimized meta tags and semantic HTML
- ⚡ **Fast Performance** - Optimized images and code splitting

## 🛠️ Tech Stack

- **Frontend Framework:** React 18
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel
